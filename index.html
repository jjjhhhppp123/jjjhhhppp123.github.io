Installation Guide · 安裝教學

Load and paste blueprints in seconds. Install once, use forever.
安裝一次，永久使用。幾秒內載入藍圖並貼上至任何位置。

INSTALL Installation Steps · 安裝步驟
01

Download Fabric Loader · 下載 Fabric Loader

Install the Fabric mod loader first.
先安裝 Fabric 模組載入器。

🔗 fabricmc.net/use/installer
02

Download Fabric API · 下載 Fabric API

Required library for most Fabric mods.
大多數 Fabric 模組需要的基礎函式庫。

🔗 modrinth.com/mod/fabric-api
03

Download WorldEdit for Fabric · 下載 WorldEdit

Get the Fabric version of WorldEdit.
下載 Fabric 版 WorldEdit。

🔗 modrinth.com/plugin/worldedit
04

Place .jar Files · 放入 .jar 檔案

Move all downloaded .jar files into your mods folder, then launch the game.
將所有 .jar 檔放入 mods 資料夾後啟動遊戲。

.minecraft/mods/
USE How to Paste a Blueprint · 如何貼上藍圖
01

Place the .schem File · 放入 .schem 檔案

Put your downloaded .schem file into the schematics folder.
將下載的 .schem 檔案放入指定資料夾。

.minecraft/config/worldedit/schematics/
02

Load the Schematic · 載入藍圖

In-game, type the command below (replace "filename" with your file's name).
進入遊戲後輸入以下指令（替換 filename 為檔案名稱）。

//schem load filename
03

Paste the Build · 貼上建築

Stand where you want the build to appear, then run:
站在想貼上的位置後輸入：

//paste
